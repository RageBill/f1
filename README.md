# F1

A simple project displaying F1 data from public [Ergast Developer API](http://ergast.com/mrd/).

## Prerequisites

- [`nodejs`](https://nodejs.org/en/)
- [`yarn`](https://yarnpkg.com/getting-started/install)

## How to use

Install the dependencies and run:

```sh
yarn install
yarn dev
```

Running tests:

```sh
yarn test
```
